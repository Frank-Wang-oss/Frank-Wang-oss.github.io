## Research Projects

### Scalable Urban Temperature Mapping via Multi-Modal Graph-based Learning
**Co-PI** (SGD 1.99M in total, SGD 757K as Co-PI) | Apr 2026 - Present

This project develops a graph-based framework to model urban temperature by integrating static urban features (e.g., building density, vegetation, sky view) and dynamic land surface temperature data. Key contributions include:

- **Spatio-temporal Graph for Temperature Prediction**: Develop advanced graph neural networks to capture complex spatial-temporal patterns in urban temperature data
- **Continual Learning for Spatial Model Adaptation**: Enable efficient model updates as new urban data becomes available by adopting a continual learning strategy to retain previously learned knowledge
- **Multi-modal Data Integration**: Fuse heterogeneous data sources including satellite imagery, weather stations, and urban infrastructure data
